# Icons Guide

```
public/icons/
├── food/          # Ikony dla jedzenia (kalorii)
│   ├── light-snack.png

│   ├── pasta.png

└── drinks/        # Ikony dla napojów (wody)
    ├── glass-water.png

    ├── large-bottle.png
    ├── cup-tea.png
    ├── mug-coffee.png

```

## Instrukcje

1. **Umieść ikony** w odpowiednich folderach zgodnie z powyższą strukturą
2. **Format**: Zalecane PNG lub SVG
3. **Rozmiar**: Optymalne 128x128px lub 256x256px
4. **Tło**: Przezroczyste tło (PNG z alpha channel)
5. **Styl**: Preferowane proste, flat design ikony

## Źródła darmowych ikon

- [Flaticon](https://www.flaticon.com/) - darmowe ikony (sprawdź licencję)
- [Icons8](https://icons8.com/) - darmowe ikony
- [Noun Project](https://thenounproject.com/) - ikony w stylu minimalistycznym
- [Freepik](https://www.freepik.com/) - darmowe grafiki

## Uwagi

- Jeśli ikona nie zostanie znaleziona, przycisk będzie dalej działać bez ikony
- Nazwy plików muszą dokładnie zgadzać się z tymi w kodzie
- Możesz dodawać własne ikony zmieniając ścieżki w pliku `dashboard/page.tsx`
